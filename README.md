# LiteCalendar
A simple and responsive Mini Calendar App built with HTML, CSS, and JavaScript.
It automatically displays today’s date, day, month, and year in a clean UI.

## Features

 Auto Updates – Shows current date, day, month, and year.

 Modern UI – Styled with gradient background and card layout.

 Responsive Design – Optimized for desktop and mobile screens.

 Lightweight – Simple, fast, and minimal code.


## Tech Stack

HTML5

CSS3 (Media Queries for responsiveness)

JavaScript (ES6)
